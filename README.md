# 🧮 My Calculator

A warm, notebook-style calculator built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

## ✨ Features

- **Notebook aesthetic** — lined paper, spiral binding, sticky-tape display, handwritten fonts
- **Tape history** — shows a running list of past calculations above the display
- **Full operations** — add, subtract, multiply, divide, percent, sign toggle
- **Keyboard support** — use number keys, operators, Enter, Backspace, Escape
- **Error handling** — division by zero shows a friendly error with a shake animation

## 📁 Project Structure

```
calculator/
├── index.html        # Main page
├── css/
│   └── style.css     # All styles (notebook theme)
├── js/
│   └── calculator.js # Calculator logic
└── README.md
```

## 🚀 Getting Started

Just open `index.html` in any browser — no build step, no server needed.

```bash
git clone https://github.com/YOUR_USERNAME/calculator.git
cd calculator
open index.html   # macOS
# or just double-click index.html on Windows/Linux
```

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0–9` | Input digit |
| `+ - * /` | Operators |
| `Enter` or `=` | Calculate |
| `Backspace` | Delete last digit |
| `Escape` | Clear all |
